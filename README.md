# Score highest on Buddymeter Quiz without playing the quiz

#### Basic Requirements:
* Python 3
* Selenium (can be installed with ```pip install selenium```)
* Google Chrome Browser

to downlaod the latest chromedriver . head over to the oofcial website : https://chromedriver.chromium.org/downloads

## How to use
- Download this repo as zip and then extract it
- go to the folder of this repo open the buddymeter_auto_score.py file. and add your name, your desired score and the buddymeter quiz link you want play.
- open a command prompt and then type ```python location_of_this_repo/buddymeter_auto_score.py```
